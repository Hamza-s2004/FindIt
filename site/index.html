About the project

FindIt is a lost-and-found Android application that lets users report, search, and manage lost or discovered items.

Package: com.example.findit

Tech stack

  • Language: Kotlin
  • Build: Gradle with Kotlin DSL
  • UI: AndroidX AppCompat, Material Components, ConstraintLayout, RecyclerView
  • compileSdk: 36 · targetSdk: 36 · minSdk: 24

How to run locally

  1. Install Android Studio.
  2. Open this folder as an existing Gradle project.
  3. Let Gradle sync, then run on an emulator or device.

From the command line with the Android SDK installed:

./gradlew assembleDebug
